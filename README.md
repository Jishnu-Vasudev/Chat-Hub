# Chat-Hub

Chat-Hub is a Python-based real-time communication system with a planned extension toward integrating AI-driven conversational capabilities. The project demonstrates the implementation of a client-server architecture for multi-user messaging and lays the foundation for a unified platform combining human communication with intelligent chatbot interaction.

# Tech Stack

Language: Python
Planned Integration: Gemini API (for AI chatbot)

# Project Structure

server.py – Manages client connections and message distribution
client.py – Handles user interaction and communication with the server

# Current Scope

Operates on localhost (single-machine environment)
Command-line interface (no GUI yet)
AI chatbot and unified interface are under development

# Future Enhancements

Integration of AI chatbot (Gemini API) for hybrid communication
Development of a GUI-based interface for improved usability
Deployment of server for cross-network communication
Addition of authentication and user session management
